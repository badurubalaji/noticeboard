# ✏️ Change the numbers in a table

⏱️ This takes about 1 minute.

This is the most common job. The TV is showing yesterday's numbers and you want to update them for today.

## When to use this guide

✅ The numbers are typed directly into NoticeBoard (you change them by hand).
❌ The numbers come from a file or website automatically. (If so, see [Upload a new JSON file](./03-upload-json.md) or [Connect to a URL](./07-connect-url.md) instead.)

---

## Step-by-step

### 1. Open the board

- Sign in to NoticeBoard.
- On the left menu, click **Boards**.
- Click the board you want to change (e.g. "Warehouse — Daily Report").
- Click the **Design** button (top right). A grid of widgets appears.

### 2. Click the table widget

You'll see the table. Click anywhere on it. A panel opens on the right showing all its settings.

### 3. Update the values

Scroll down in the right panel to the section called **Table Config**.

You'll see a grid that looks like the table on the TV. Each box (called a "cell") has a value — click any of them to edit:

| Name   | Today | Yesterday |
|--------|------:|----------:|
| Line A |   142 |       138 |
| Line B |    89 |        95 |
| Line C |   201 |       211 |

- Click any cell.
- Type the new number.
- Press **Tab** or click the next cell.

### 4. Save

Click the **Save** button (top right). You'll see a brief spinner, then it's saved.

### 5. Check the TV

The TV updates within a few seconds — you don't need to walk to it. If you're sitting at the TV's URL on your computer, just wait a moment.

✅ **Done!**

---

## Quick things you might want to do

### Add a new row
At the bottom of the table editor, click **+ Add Row**. A blank row appears. Fill it in.

### Remove a row
Each row has a small **✕** on the right. Click it.

### Add a new column
Above the rows, click **+Col**. A blank column appears.

### Change a column heading
At the top of the table editor, click the heading text and type the new name.

### Show totals
In the right panel, find the **Totals** section.

- Tick **"Show totals row"** — adds a row at the bottom that adds up each column.
- Tick **"Show totals column"** — adds a column on the right that adds up each row.

NoticeBoard only adds up cells that contain numbers. Cells with words are skipped, so "Line A | 142 | 138" still totals to 280.

---

## What if I made a mistake?

- **Just changed something wrong?** Click the cell and re-type the correct value, then **Save** again.
- **Already saved?** Same fix. Change it back, save again.
- **Want to undo a lot of changes?** Press **Ctrl + Z** (Windows) or **⌘ + Z** (Mac) *before* you save. After you save, you'll need to type the values back manually.

---

## I don't see a table when I click — only a chart / image / something else

That widget isn't a table. Either:

- Click somewhere else on the page to find the right widget, or
- Look at the right-side panel — the heading tells you what kind of widget you've selected ("Table Config", "Chart Config", "Image", etc.).
