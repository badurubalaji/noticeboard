## A few words you'll see in NoticeBoard

You don't need to memorise these — just glance back here if you get stuck.

| Word | What it means in plain English |
|---|---|
| **Board** | One screen worth of information. Each TV shows one board. |
| **Page** | A board can flip between pages. Like slides in a presentation. |
| **Widget** | A box on a page that shows one thing (a table, a chart, an image…). |
| **Template** | A ready-made design. You fill in the blanks. |
| **Data source** | A connection to your numbers — either a website (URL) or a file (JSON) you upload. |
| **Display** | The TV/tablet view that staff see. The thing you're *making* with NoticeBoard. |
